# 上級 問題05: 画像ライトボックス

**難易度: ★★★★★★★☆☆☆**

## 🎯 やること

サムネイルをクリックすると、**画面全体に大きい画像**が表示される「ライトボックス」を作ります。

## ✅ 要件

1. 6枚のサムネイルをグリッド表示
2. サムネイルをクリックすると、画面全体の半透明オーバーレイ + 中央に大きな画像
3. ライトボックス表示中に「×」ボタン、**← →** ボタンで前後の画像に切り替え
4. オーバーレイ背景クリックで閉じる
5. Esc で閉じる、← → キーでも前後へ
6. 最後の画像で「次へ」を押したら最初に戻る（ループ）

```js
let currentIndex = 0;
const urls = [...];
```
イメージのクリックで index を特定、ライトボックスを開いて URL を設定。
