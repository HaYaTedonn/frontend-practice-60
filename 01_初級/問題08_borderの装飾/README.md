# 初級 問題08: border の装飾

**難易度: ★★★☆☆☆☆☆☆☆**

## 🎯 やること

要素に**枠線（border）**を付けて、いろいろな装飾を試してみましょう。

## ✅ 要件

`style.css` を編集して、次の通りに設定してください。

1. `.box1` に **実線** の青い枠線（太さ 2px）を付ける
2. `.box2` に **点線** のオレンジの枠線（太さ 4px）を付ける
3. `.box3` に **二重線** の緑の枠線（太さ 6px）を付ける
4. すべての `.box` クラスに次も適用する
   - 内側余白（padding）: `16px`
   - 外側余白（margin）: `12px`
   - 幅（width）: `250px`
   - 角を丸める → `border-radius: 8px`

## 👀 確認方法

3つのボックスが縦に並び、それぞれ別の形式の枠線が付いていて、角が丸くなっていればOK。

## 💡 ヒント

- `border: 太さ 種類 色;` の形で一行で書ける
  - 例: `border: 2px solid blue;`
- 種類: `solid`（実線）/ `dotted`（点線）/ `dashed`（破線）/ `double`（二重）
- 角を丸める: `border-radius: 8px`
