# 初級 問題02: 見出しと段落

**難易度: ★☆☆☆☆☆☆☆☆☆**

## 🎯 やること

`<h1>` 〜 `<h6>` の**見出し**と、`<p>` の**段落**を使って、自己紹介ページを作ってみましょう。

## ✅ 要件

1. `<h1>` で「自己紹介」と大きな見出しを書く
2. `<h2>` で「名前」と書き、その下の `<p>` に自分の名前（偽名OK）を書く
3. `<h2>` で「趣味」と書き、その下の `<p>` に趣味を書く
4. `<h3>` で「好きな食べ物」と書き、その下の `<p>` に好きな食べ物を書く

## 👀 確認方法

ブラウザで開いたときに、見出しが階層的に大きさが変わっていて、段落がその下に表示されていれば正解です。

## 💡 ヒント

- `<h1>` が一番大きく、`<h6>` が一番小さい
- `<p>` は **p**aragraph（段落）の略
- 1ページに `<h1>` は **1つだけ**にするのが基本ルール
