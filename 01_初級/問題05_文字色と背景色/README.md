# 初級 問題05: 文字色と背景色

**難易度: ★★☆☆☆☆☆☆☆☆**

## 🎯 やること

CSS を使って、**文字の色**と**背景の色**を変えてみましょう。

## ✅ 要件

`style.css` を編集して、次のようなスタイルを適用してください。

1. `<body>` の**背景色**を `lightyellow` にする
2. `<h1>` の**文字色**を `crimson`（深い赤）にする
3. `<p>` の**文字色**を `darkslategray` にする
4. `<p>` の**背景色**を `white` にする

## 👀 確認方法

ブラウザで開くと、
- ページ全体がうっすら黄色い
- 見出しが赤く表示される
- 段落の文字が濃い緑っぽい灰色で、白背景になっている

## 💡 ヒント

- 文字色 → `color`
- 背景色 → `background-color`
- 色の書き方: `red`（色名）/ `#ff0000`（16進数）/ `rgb(255,0,0)`
