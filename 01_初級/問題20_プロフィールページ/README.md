# 初級 問題20: プロフィールページ

**難易度: ★★★★★☆☆☆☆☆**

## 🎯 やること

初級の集大成として、**プロフィールページ**を一枚作りましょう。

## ✅ 要件

次の構成のページを作ってください。

1. **ヘッダー**（`<header>`）
   - 中央に「John Doe のプロフィール」というタイトル
   - 背景色 `#0f172a`、文字色 白、高さ 80px、文字 24px

2. **プロフィール画像**
   - 円形（`border-radius: 50%`）、120×120
   - ページ中央に配置

3. **名前**（h1）、**職業**（p）を中央揃えで

4. **自己紹介エリア**（section）
   - 背景色 `#f3f4f6`、角丸 12px、padding 20px、最大幅 600px、中央寄せ
   - `<h2>自己紹介</h2>` と `<p>` 3行程度

5. **スキル一覧**（ul）
   - 3〜5個をバッジ風に表示（横並び、背景色付き、角丸）

6. **フッター**（`<footer>`）
   - `© 2026 John Doe` と中央揃えで表示

構成は自由にアレンジして構いません（全体が**それっぽいプロフィールページ**になればOK）。
