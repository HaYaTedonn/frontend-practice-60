# 初級 問題18: カード型 UI

**難易度: ★★★★★☆☆☆☆☆**

## 🎯 やること

商品紹介などでよく使う**カード型 UI**を作ります。これまでの知識の総動員です。

## ✅ 要件

1 枚のカードに次を含めてください。

1. 上部に**画像**（幅100%、高さ 180px、`object-fit: cover`）
2. その下に**タイトル**（`<h3>`）
3. その下に**説明文**（`<p>`）
4. 一番下に**ボタン**（「詳しく見る」）

CSS で：
- カード全体: 幅 280px、白背景、角丸 12px、影付き、`overflow: hidden`
- タイトル / 説明 / ボタンは `padding: 16px` の中に収める
- ボタン: 青系、白文字、角丸、hover で色が変わる
- **カード全体に hover で少し浮き上がる効果**（`transform: translateY(-4px)` + 影が強くなる）
- `transition` で滑らかに
