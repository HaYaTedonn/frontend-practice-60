# 初級 問題09: テーブルを作る

**難易度: ★★★☆☆☆☆☆☆☆**

## 🎯 やること

`<table>` タグで、**3行3列**の成績表を作ってみましょう。

## ✅ 要件

1. `<table>` タグでテーブルを作る
2. 1行目は見出し行（`<th>`）で、「教科」「点数」「評価」
3. 2〜4行目は普通のセル（`<td>`）で以下のデータを入れる
   - 国語 / 80 / B
   - 数学 / 95 / A
   - 英語 / 70 / C
4. CSS で次を装飾する
   - テーブル全体に `border-collapse: collapse;` を付けて枠線を重ねる
   - `th` と `td` に `1px solid #333` の枠線を付ける
   - `th` と `td` に `8px 16px` の padding を付ける
   - `th` の背景色を `lightgray` にする

## 👀 確認方法

成績表らしい見た目（枠線が付いて、見出しだけ背景色が違う）になっていれば正解。

## 💡 ヒント

- 行 → `<tr>` （table row）
- 見出しセル → `<th>` （table header）
- データセル → `<td>` （table data）
- `border-collapse: collapse` で枠線が二重にならない
