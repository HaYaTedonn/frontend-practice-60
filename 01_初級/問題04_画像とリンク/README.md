# 初級 問題04: 画像とリンク

**難易度: ★★☆☆☆☆☆☆☆☆**

## 🎯 やること

`<img>` タグで**画像を表示**し、`<a>` タグで**他のページへのリンク**を作ってみましょう。

## ✅ 要件

1. ページ上部に `<h1>お気に入り紹介</h1>` を書く
2. 以下の URL の画像を `<img>` タグで表示する（好きな画像URLでもOK）
   - URL: `https://placehold.jp/300x200.png`
   - 代替テキスト（`alt`）に「サンプル画像」と指定する
3. 画像の下に、Google (`https://www.google.com`) へのリンクを `<a>` タグで作る
   - リンクのテキストは「Googleで検索する」
   - **新しいタブで開く**ように設定する

## 👀 確認方法

- 画像が表示される
- 「Googleで検索する」をクリックすると、新しいタブで Google が開く

## 💡 ヒント

- `<img>` は閉じタグがいらない: `<img src="画像URL" alt="代替テキスト">`
- `<a>` は囲むタグ: `<a href="URL">リンク文字</a>`
- 新しいタブで開く → `target="_blank"` を追加
