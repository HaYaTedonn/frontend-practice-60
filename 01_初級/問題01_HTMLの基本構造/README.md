# 初級 問題01: HTMLの基本構造

**難易度: ★☆☆☆☆☆☆☆☆☆**

## 🎯 やること

すべての HTML ファイルに必要な**基本構造**を書いてみましょう。

`index.html` を編集して、以下の条件を満たすようにしてください。

## ✅ 要件

1. 1行目に `<!DOCTYPE html>` を書く
2. `<html>` タグで全体を囲む（`lang="ja"` を指定）
3. `<head>` タグの中に以下を入れる
   - 文字コード指定: `<meta charset="UTF-8">`
   - タイトル: `<title>はじめてのページ</title>`
4. `<body>` タグの中に `<h1>Hello, World!</h1>` を書く

## 👀 確認方法

`index.html` をブラウザで開くと、ブラウザのタブに「はじめてのページ」、画面に「Hello, World!」と表示されれば正解です。

## 💡 ヒント

- タグは必ず**開きタグ** `<html>` と**閉じタグ** `</html>` をセットで書く
- `<meta>` のような**自己完結するタグ**は閉じタグがいらない
