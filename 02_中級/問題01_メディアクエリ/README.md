# 中級 問題01: メディアクエリ（レスポンシブ基礎）

**難易度: ★★★☆☆☆☆☆☆☆**

## 🎯 やること

**画面サイズに応じてレイアウトを切り替える**メディアクエリを使ってみましょう。

## ✅ 要件

3つのカードが用意されています。次の挙動になるようにしてください。

1. **画面幅 769px 以上** のとき
   - `.cards` が `display: flex; gap: 20px;` で**横並び**
   - 各カードの幅は `1fr`相当（`flex: 1`）

2. **画面幅 768px 以下** のとき
   - `.cards` が `flex-direction: column` で**縦並び**
   - 各カードが幅 100%

3. **画面幅 480px 以下** のとき
   - 背景色を `#fff7ed` に変え、カードの padding を狭く（12px）する

## 👀 確認方法

ブラウザの幅を変えて、レイアウトが切り替わるかチェック。開発者ツール（F12）のデバイスモードでも確認できます。

## 💡 ヒント

```css
@media (max-width: 768px) {
  .cards { flex-direction: column; }
}
```
