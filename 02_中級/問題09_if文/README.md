# 中級 問題09: if 文による条件分岐

**難易度: ★★★★★☆☆☆☆☆**

## 🎯 やること

入力された年齢に応じて、メッセージや色を**分岐**させて表示します。

## ✅ 要件

年齢入力欄 (`#age`) と判定ボタン (`#judgeBtn`) があります。ボタンを押したときの挙動：

1. 年齢が**未入力（空文字や NaN）**の場合は、`.result` に「年齢を入力してください」と表示し、文字色を `gray` にする
2. 年齢が **20 以上**なら「成人です」を青色 (`navy`) で表示
3. 年齢が **13 以上 20 未満**なら「未成年です」を緑色 (`green`) で表示
4. 年齢が **0 以上 13 未満**なら「子どもです」を橙色 (`darkorange`) で表示
5. 負の値が入力されたら「正しい年齢を入力してください」を赤色 (`red`) で表示

## 👀 確認方法

- 各境界値（0, 12, 13, 19, 20, -1, 空欄）で動作を確認

## 💡 ヒント

```js
const age = Number(input.value);
if (Number.isNaN(age) || input.value === '') {
  ...
} else if (age < 0) {
  ...
} else if (age < 13) {
  ...
}
```
