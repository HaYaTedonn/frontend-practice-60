# 中級 問題17: フォームの入力チェック

**難易度: ★★★★★★☆☆☆☆**

## 🎯 やること

フォーム送信前に、入力値を JavaScript で**バリデーション（検証）**します。

## ✅ 要件

ログインフォームに対して、以下のチェックを行ってください。

1. 「お名前」が空なら、その下のエラー表示に「お名前を入力してください」を赤色で表示
2. 「メール」が空なら「メールアドレスを入力してください」、形式不正なら「メール形式が正しくありません」を表示
   - 簡易チェック：`@` を含むこと
3. 「パスワード」が 8 文字未満なら「8文字以上で入力してください」を表示
4. **すべて OK** の場合のみ alert で「送信成功！」と表示
5. フォーム送信時は**ページを再読み込みさせない**（`event.preventDefault()`）
6. エラーがあれば、該当 input に `.invalid` クラスを付け、赤い枠を出す（CSS 済み）

## 💡 ヒント

```js
form.addEventListener('submit', (e) => {
  e.preventDefault();
  // 各 input を検証
});
```
