# 中級 問題04: JSの基本（変数と alert）

**難易度: ★★★★☆☆☆☆☆☆**

## 🎯 やること

初めての JavaScript！ **変数**に値を入れて、**alert**で表示してみましょう。

## ✅ 要件

`script.js` に次のコードを書いてください。

1. `const name = "あなたの名前";` のように**文字列**を変数に入れる
2. `const age = 20;` のように**数値**を変数に入れる
3. `alert(...)` で「`私の名前は XXX、年齢は XX です。`」と表示する
   - 変数を埋め込むには**テンプレートリテラル**（バッククォート \`...\`）を使う
4. `console.log(...)` でも同じ文字列を出力する

## 👀 確認方法

- ページを開くと alert ダイアログが出る
- 開発者ツール（F12）の Console タブにも同じ文字列が出る

## 💡 ヒント

```js
const message = `私の名前は ${name}、年齢は ${age} です。`;
alert(message);
console.log(message);
```
