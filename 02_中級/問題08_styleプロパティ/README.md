# 中級 問題08: style プロパティで装飾変更

**難易度: ★★★★★☆☆☆☆☆**

## 🎯 やること

JavaScript から**要素のスタイルを直接変更**してみます。class とは違う使いどころを知りましょう。

## ✅ 要件

1. 文字サイズを変えるレンジスライダー（`#sizeRange`）で、`.target` 段落の `font-size` が**リアルタイム**に変わる
   - スライダーの値 (px) → 段落の文字サイズ
2. カラーピッカー（`#colorPicker`）で、`.target` の文字色が即座に変わる
3. 「太字トグル」ボタンで、太字⇔通常を切り替える
   - スタイルを直接切り替える方法で実装（`el.style.fontWeight`）

## 👀 確認方法

- スライダーを動かすと文字が大きくなる・小さくなる
- カラーピッカーで好きな色に
- トグルボタンで太字のオンオフ

## 💡 ヒント

- `input` イベントで値の変化をリアルタイムに拾える
```js
sizeRange.addEventListener('input', (e) => {
  target.style.fontSize = `${e.target.value}px`;
});
```
