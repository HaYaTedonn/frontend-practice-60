# 中級 問題16: 画像スライダー（手動）

**難易度: ★★★★★★☆☆☆☆**

## 🎯 やること

「前へ / 次へ」ボタンで画像が切り替わる**手動スライダー**を作ります。

## ✅ 要件

1. 3枚の画像が用意されている。初期表示は最初の画像
2. 「次へ」ボタンで次の画像へ、「前へ」で前の画像へ
3. 先頭で「前へ」を押したら最後の画像へ（ループ）、末尾で「次へ」を押したら最初に戻る
4. 現在の画像番号（1 / 3 のような形）を表示
5. ドットインジケータ（3つの丸）を表示し、現在位置の丸に `.active` クラス
6. 各ドットをクリックするとその画像へジャンプ

## 💡 ヒント

```js
let current = 0;
const images = [...];
function show(i) {
  // i 番目を表示、他を隠す
}
```

ループの計算は `(current + 1) % length` が便利。
