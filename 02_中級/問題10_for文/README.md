# 中級 問題10: for 文で繰り返し

**難易度: ★★★★★☆☆☆☆☆**

## 🎯 やること

`for` 文で繰り返し処理を行い、DOM に動的に要素を作り出します。

## ✅ 要件

ボタン `#genBtn` を押したら、次を `.list` に生成してください。

1. 1 から 10 までの数字を `<li>` に入れて表示
2. **偶数**の `<li>` には `.even` クラスを付ける（CSS で背景色が変わる）
3. `<li>` のテキストは `1番目 / 2番目 / ...` とする
4. 既に生成済みの内容は、毎回クリアしてから追加する（`.list.innerHTML = ''`）

## 👀 確認方法

- ボタンを押すと 1〜10 が表示
- 偶数番目が色付きになる
- 何度押しても表示が重複しない

## 💡 ヒント

```js
for (let i = 1; i <= 10; i++) {
  const li = document.createElement('li');
  li.textContent = `${i}番目`;
  list.appendChild(li);
}
```
